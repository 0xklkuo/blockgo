# Code of Conduct

## Our Commitment

We want BlockGo to be a welcoming, respectful, and educational community for developers of all levels.

## Expected Behavior

Please:

- be respectful and constructive
- assume good intent
- give actionable technical feedback
- welcome newcomers
- focus on the code and ideas, not the person
- keep discussions professional and inclusive

## Unacceptable Behavior

Examples include:

- harassment or insulting language
- personal attacks
- discriminatory behavior
- trolling or deliberate disruption
- publishing private information without permission

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and other community interactions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or contributions that do not align with this Code of Conduct.
